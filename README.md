# 🛒 Retail Analytics SQL Project

## 📌 Overview

This project focuses on analyzing retail sales data using SQL. It includes data cleaning, handling inconsistencies, and generating business insights.

## 🧰 Tools Used

* MySQL
* SQL

## 🔧 Key Steps Performed

* Data Cleaning (NULL handling, duplicates removal)
* Price discrepancy correction
* Date format conversion
* Data validation checks

## 📊 Business Analysis

* Top selling products
* Customer purchase frequency
* Category performance
* Sales trend analysis
* Month-on-Month growth
* Customer segmentation

## 💡 Key Insights

* Identified top revenue-generating products
* Found duplicate transactions affecting data accuracy
* Analyzed customer behavior and purchase patterns
