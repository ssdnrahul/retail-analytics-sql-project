# 🛒 Retail Analytics SQL Project

## 📌 Overview

This project focuses on analyzing retail sales data using SQL. It includes data cleaning, handling inconsistencies, and generating business insights from transactional data.

This is a SQL-focused project and does not include dashboards or visualization tools.

---

## 🧰 Tools Used

* MySQL
* SQL

---

## 🔧 Data Cleaning & Preparation

* Fixed column name encoding issues
* Handled NULL and blank values
* Removed duplicate transactions
* Corrected price discrepancies between tables
* Converted date column from TEXT to DATE format
* Performed data validation checks

---

## 📊 Business Analysis

* Total sales and quantity per product
* Customer transaction frequency
* Category-wise performance analysis
* Top 10 revenue-generating products
* Lowest selling products
* Daily sales trend analysis
* Month-on-Month (MoM) growth calculation

---

## 👥 Customer Analysis

* High-value customers identification
* Low-frequency customer analysis
* Repeat purchase behavior
* Customer loyalty analysis (purchase duration)

---

## 🎯 Customer Segmentation

* Segmented customers into Low, Medium, and High categories
* Counted number of customers in each segment

---

## 📁 Project Structure

* `retail_analysis.sql` → Main SQL project file

---

## 🚀 How to Run

1. Import dataset into MySQL
2. Execute the SQL script step-by-step
3. Analyze query outputs

---

## 💡 Key Insights

* Identified top-performing products contributing maximum revenue
* Found duplicate transactions affecting data accuracy
* Observed variations in customer purchase behavior
* Analyzed sales trends and growth patterns over time
